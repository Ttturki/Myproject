# Endoscopy Unit Dashboard

A web application for managing patient transfers in an endoscopy unit. Features a table-based dashboard with editable fields for patient details.

## Features
- View and edit patient information (name, file number, arrival time, etc.).
- Color-coded rows based on patient location.
- Responsive design for desktop and mobile.

## How to Run
1. Clone the repository: `git clone <repository-url>`
2. Open `index.html` in a web browser.

## File Structure
- `index.html`: Main HTML file
- `styles.css`: Stylesheet
- `script.js`: JavaScript logic
- `images/`: Optional folder for images (e.g., logo)
